#MasterWordle

MasterWordle è un word game ispirato a Wordle con modalità multiple, statistiche salvate e difficoltà progressive. Indovina la parola segreta nel minor numero di tentativi possibile.

Il gioco include:
- Modalità 4, 5 e 6 lettere
- Sistema punti e statistiche salvate localmente
- Tema dark/light dinamico
- Tastiera virtuale responsive
- Condivisione risultati stile Wordle
- Compatibile desktop e mobile

È collegato alla repository centrale MasterHub (https://mastersabba.github.io/MasterSabba/), piena di altri minigame.
